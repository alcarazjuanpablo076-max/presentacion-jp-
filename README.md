# Presentación JP

Este es el sitio web personal de **Juan Pablo "JP" Alcaraz Mesina**.

## Descripción
Una página de presentación "one-page" con un diseño limpio, moderno y amigable. 
Incluye:
- Sección Hero con saludo.
- Contexto personal y deportivo.
- Galería de fotos (collage).
- Reproductor de música de fondo.

## Tecnologías
- HTML5
- CSS3 (Variables CSS, Flexbox, Grid, Animaciones)
- JavaScript (Control de audio)
